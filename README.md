### 自动汇总网盘链接

汇总所有网盘链接与提取码，按照顺序一一对应。汇总面板位于右侧边栏。

### 自动填充百度网盘提取码

汇总后的网盘链接自动填充提取码。
支持百度网盘 (pan.baidu.com) 与蓝奏云 (lanzn.com) 网盘。
